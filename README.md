# Excel-Stock-Market-Dashboard.
A dynamic Excel-based analytics dashboard designed to track and visualize stock market trends. This project utilizes advanced Excel functions like INDEX/MATCH for real-time data retrieval, Data Validation for interactive user controls, and automated charting to compare historical performance (e.g., D-Mart vs. Asian Paints).

Excel Stock Market Analytics Dashboard
## Situation
In the fast-paced world of financial trading, monitoring multiple stock performances manually using static spreadsheets is inefficient and prone to error. I needed a way to visualize historical data and extract key performance indicators (KPIs) dynamically for companies like D-Mart and Asian Paints to make data-driven investment comparisons.

## Task
The goal was to build a professional, interactive portfolio project that automates the analysis of one year of stock market data. I aimed to create a system where a user could select a stock from a dropdown menu and instantly see updated charts and financial metrics without manually filtering the raw data.

## Action
To solve this, I implemented the following technical solutions:

Data Acquisition: Extracted comprehensive historical stock data (Open, High, Low, Close, Volume) from Yahoo Finance.

Dynamic Logic: Engineered complex formulas using INDEX and MATCH to create a search engine within Excel that retrieves specific data points based on the current system date and selected stock.

Interactive UI: Developed a Data Validation system to create a user-friendly dropdown interface.

Advanced Visualization: Created dynamic charts that auto-adjust their scales and data series based on user selection, allowing for side-by-side comparison of different stock volumes and price trends.

## Result
I successfully developed a fully automated Stock Market Dashboard that reduces analysis time by eliminating manual data entry.

Efficiency: Users can toggle between multiple stocks in a single click to view 52-week highs/lows and daily trends.

Accuracy: The use of dynamic formulas ensures that as new data is added, the dashboard updates automatically without breaking the visual reports.

Portfolio Ready: The final product demonstrates high proficiency in Excel-based data analytics and financial modeling.
